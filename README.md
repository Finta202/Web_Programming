# Web Programming – Filmovi i Serije

Railway link to site: 
https://webprogramming-production-baea.up.railway.app/

Aplikacija omogućuje pregled popisa filmova i serija, filtriranje podataka prema različitim kriterijima te dodavanje odabranih naslova u košaricu za posudbu.


## Funkcionalnosti aplikacije

## Dohvat i prikaz podataka
- Učitavanje podataka iz `movies.csv` datoteke
- Parsiranje CSV datoteke pomoću PapaParse biblioteke
- Dinamički prikaz filmova i serija u HTML tablici

### Filtriranje i pretraživanje
Mogućnost filtriranja prema sljedećim kriterijima:

- Pretraga po nazivu filma
- Filtriranje po žanru
- Raspon godina (od – do)
- Filtriranje po trajanju
- Minimalna ocjena

### Košarica
- Dodavanje filmova u košaricu
- Uklanjanje filmova iz košarice
- Potvrda posudbe filmova
